# TypeScript 进阶 20 题（详细版）

## 1) 条件类型
### 代码示例
```ts
type IsString<T> = T extends string ? true : false;
```
### 原理讲解
按类型关系进行分支推导，是高级类型基础。
### 练习
- 基础练习：写一个 `IsArray<T>`。
- 进阶练习：补一个边界场景（空值/并发/错误处理）并解释。
### 高频追问

### 什么时候会触发分布式？

当 T 是联合且裸类型参数出现在条件左侧时。

### 技术要点总结
条件类型是类型计算的 if/else。

### 技术要点总结
