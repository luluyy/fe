# 学习路线（4 周）

## 第 1 周：JavaScript 基础

- 作用域与闭包
- 原型链与继承
- 事件循环（宏任务/微任务）
- this 绑定与调用方式

## 第 2 周：浏览器与网络

- 缓存（强缓存/协商缓存）
- Cookie / Session / JWT
- localStorage / sessionStorage / IndexedDB
- 从输入 URL 到页面展示

## 第 3 周：React 与工程化

- React 渲染流程与状态管理
- Hook 常见陷阱
- 打包与构建优化
- 性能排查思路

## 第 4 周：项目与系统设计

- 项目亮点提炼
- 权衡与取舍
- 故障与复盘案例
