# AI 项目实战场景 10 题（详细版）

## 1) 设计企业知识问答系统
### 代码示例
```txt
ingest -> chunk -> embed -> index -> retrieve -> rerank -> generate
```
### 原理讲解
核心是检索质量与答案可追溯。
### 练习
- 基础练习：列组件清单。
- 进阶练习：给出 SLA 目标。
### 高频追问
### 如何保证回答可追溯？
输出必须带来源片段和文档 ID。
### 面试口述模板
架构要围绕质量、可追溯、成本与运维闭环。

## 2) RAG 命中率下降排查
### 代码示例
```txt
check: parser/chunk/embed/index/rerank/prompt
```
### 原理讲解
逐层排查比盲目调参数更高效。
### 练习
- 基础练习：写排查顺序。
- 进阶练习：设计自动巡检。
### 高频追问
### 如何快速止血？
回滚索引版本或重排策略到稳定基线。
### 面试口述模板
先定位层级，再做最小可回滚修复。

## 3) 处理模型供应商不可用
### 代码示例
```ts
try A -> fallback B -> fallback cached
```
### 原理讲解
多供应商与熔断策略保证可用性。
### 练习
- 基础练习：定义降级优先级。
- 进阶练习：设计自动恢复策略。
### 高频追问
### 如何避免降级后成本暴涨？
设置预算阈值和熔断后限流。
### 面试口述模板
降级要兼顾可用性与成本，不是单纯切换模型。

## 4) 构建 AI 客服工单助手
### 代码示例
```txt
intent classify -> tool call -> draft reply -> human-in-loop
```
### 原理讲解
高风险操作必须 human-in-loop。
### 练习
- 基础练习：定义风险分级。
- 进阶练习：定义人工审核触发条件。
### 高频追问
### 如何防误操作？
把执行动作拆成“建议”和“确认”两阶段。
### 面试口述模板
AI 助手应默认“辅助决策”，关键环节保留人工确认。

## 5) 多租户知识库隔离
### 代码示例
```ts
filter = { tenantId }
```
### 原理讲解
检索层、缓存层、日志层都需租户边界。
### 练习
- 基础练习：列出隔离点。
- 进阶练习：设计审计策略。
### 高频追问
### 哪层最易漏？
缓存键和日志脱敏最容易漏租户维度。
### 面试口述模板
多租户隔离必须全链路一致，单点隔离不够。

## 6) 提升首字延迟 TTFT
### 代码示例
```txt
prefetch context + stream response
```
### 原理讲解
TTFT 受检索、模型首 token、网络链路共同影响。
### 练习
- 基础练习：给出 3 个优化点。
- 进阶练习：定义 TTFT 监控看板。
### 高频追问
### 为什么总延迟下降但体感仍差？
TTFT 没改善，用户仍觉得“卡住”。
### 面试口述模板
用户体感优先看 TTFT，再看总耗时。

## 7) 低成本上线 MVP
### 代码示例
```txt
small model + strict prompt + cache + nightly eval
```
### 原理讲解
MVP 先保证可用与可评估，再逐步提质。
### 练习
- 基础练习：给 MVP 定义验收标准。
- 进阶练习：定义升级路径。
### 高频追问
### MVP 常见误区？
忽略评测和监控，导致后续无法迭代。
### 面试口述模板
MVP 不是简化质量，而是先建立可迭代闭环。

## 8) AI 输出安全审核链路
### 代码示例
```txt
model output -> safety classifier -> policy engine -> final response
```
### 原理讲解
输出侧同样需要策略引擎与人工兜底。
### 练习
- 基础练习：设计一条违规拦截规则。
- 进阶练习：加入人工复核队列。
### 高频追问
### 为什么只做输入审核不够？
模型可能在输出阶段生成风险内容。
### 面试口述模板
安全审核要覆盖输入和输出双向。

## 9) 构建评测与回归平台
### 代码示例
```ts
runEval(dataset, promptVersion, modelVersion)
```
### 原理讲解
版本化评测能防“改一个 prompt 退一片”。
### 练习
- 基础练习：定义 case schema。
- 进阶练习：设计自动回归触发条件。
### 高频追问
### 如何降低人工评测成本？
先自动评分过滤，再对边界样本做人审。
### 面试口述模板
评测平台是 AI 持续交付的核心基础设施。

## 10) 向面试官讲你的 AI 项目
### 代码示例
```txt
背景 -> 目标 -> 架构 -> 难点 -> 结果 -> 复盘
```
### 原理讲解
结构化叙事比堆技术名词更有说服力。
### 练习
- 基础练习：写 1 分钟项目介绍。
- 进阶练习：准备 3 个追问答案。
### 高频追问
### 最难的取舍是什么？
通常是质量、延迟、成本三者平衡。
### 面试口述模板
我会用数据说明取舍：为什么这样选、结果如何、若重来会怎么做。
