AI 开发岗进阶 15 题（详细版）

1) RAG 召回差怎么排查？​

代码示例​

// 检查 query -> embedding -> topK -> rerank -> context 质量

原理讲解​

问题通常来自切分、向量模型、索引参数、重排策略。

练习​

  • 基础练习：列一个 5 步排查清单。
  • 进阶练习：设计自动回归评测集。

高频追问​

先查哪一步最省时间？​

先查检索前 10 条是否语义相关，再看重排和提示词。

面试口述模板​

RAG 排查先定位层级：召回质量、重排质量、上下文拼接、生成约束。

2) 多模型路由怎么做？​

代码示例​

const model = score < 0.5 ? 'small' : 'large';

原理讲解​

按任务复杂度、延迟预算、风险等级动态选择模型。

练习​

  • 基础练习：定义一个路由打分函数。
  • 进阶练习：设计降级链路。

高频追问​

如何防止路由抖动？​

加滞后阈值和最小稳定窗口，避免频繁切换。

面试口述模板​

多模型路由是成本与质量平衡器，核心是可解释策略和可回滚。

3) Agent 如何做状态管理？​

代码示例​

state = { goal, plan, toolHistory, memoryRef }

原理讲解​

区分短期会话状态和长期记忆，避免上下文无限膨胀。

练习​

  • 基础练习：定义一份 agent state schema。
  • 进阶练习：设计 memory 压缩策略。

高频追问​

什么时候该清理记忆？​

任务切换、超时会话、低价值历史应清理或摘要化。

面试口述模板​

Agent 状态管理要可追踪、可压缩、可恢复。

4) 工具调用安全怎么做？​

代码示例​

if (!allowList.includes(toolName)) throw new Error('forbidden');

原理讲解​

工具调用需权限校验、参数校验、审计日志三层保护。

练习​

  • 基础练习：设计一个工具调用 ACL。
  • 进阶练习：设计工具沙箱执行模型。

高频追问​

参数注入怎么防？​

对参数做 schema 校验和危险字段过滤。

面试口述模板​

工具安全不是提示词问题，而是鉴权 + 校验 + 沙箱的系统问题。

5) Prompt 版本管理怎么做？​

代码示例​

promptKey = `qa:v3:${tenantId}`;

原理讲解​

Prompt 应像代码一样版本化、可回滚、可灰度。

练习​

  • 基础练习：设计 prompt 元数据字段。
  • 进阶练习：设计灰度发布流程。

高频追问​

Prompt 改动如何评估风险？​

先离线评测，再小流量 A/B，最后全量。

面试口述模板​

Prompt 是生产资产，必须纳入版本与发布体系。

6) AI 观测性要采哪些指标？​

代码示例​

metrics = { ttft, latency, tokenIn, tokenOut, cost, successRate }

原理讲解​

无观测就无法优化质量、成本与稳定性。

练习​

  • 基础练习：设计一张 AI 请求日志表。
  • 进阶练习：定义告警阈值。

高频追问​

TTFT 和总延迟怎么权衡？​

交互产品优先 TTFT，批处理任务优先吞吐与总时长。

面试口述模板​

AI 观测要覆盖体验、质量、成本、稳定性四维。

7) 流式中断恢复怎么做？​

代码示例​

if (clientAbort) savePartial(traceId, partialText)

原理讲解​

需要幂等请求 ID、分片缓冲与可恢复上下文。

练习​

  • 基础练习：设计断点续生成协议。
  • 进阶练习：处理重复分片去重。

高频追问​

为什么会出现重复 token 片段？​

网络重传或客户端重连导致重复消费，需要序号去重。

面试口述模板​

流式恢复核心是请求幂等和分片可追踪。

8) 评测集怎么构建？​

代码示例​

case = { query, context, expected, rubric }

原理讲解​

评测集应覆盖高频、边界、对抗、失败样例。

练习​

  • 基础练习：构建 20 条最小评测集。
  • 进阶练习：定义自动评分规则。

高频追问​

为什么线上指标也会“假好”？​

用户不一定反馈错误，需离线金标与人工抽检结合。

面试口述模板​

评测集是 AI 迭代的地基，决定你优化是否可信。

9) 向量索引参数怎么调？​

代码示例​

// HNSW: M, efConstruction, efSearch

原理讲解​

参数影响召回率、延迟、内存占用三者平衡。

练习​

  • 基础练习：给一套低延迟参数建议。
  • 进阶练习：给一套高召回参数建议。

高频追问​

调参顺序建议？​

先定延迟预算，再逐步提高召回参数并评测。

面试口述模板​

索引调参本质是召回-延迟-成本三角优化。

10) 长上下文优化策略​

代码示例​

context = rerank(topK).slice(0, budgetTokens)

原理讲解​

通过重排、摘要、去重与预算裁剪控制上下文质量。

练习​

  • 基础练习：写 token 预算分配规则。
  • 进阶练习：实现上下文去重策略。

高频追问​

为什么上下文越长不一定越好？​

噪声增加会稀释关键信号并提高成本与延迟。

面试口述模板​

长上下文优化关键在“信息密度”，不是“长度最大化”。

11) A/B 实验在 AI 场景怎么做？​

代码示例​

bucket = hash(userId) % 100 < 10 ? 'B' : 'A'

原理讲解​

确保随机分流、样本量和显著性，避免偏差结论。

练习​

  • 基础练习：定义 A/B 成功指标。
  • 进阶练习：设计回滚阈值。

高频追问​

为什么需要 guardrail 指标？​

主指标提升可能牺牲延迟或成本，需设护栏防副作用。

面试口述模板​

AI A/B 不只看一个分数，要看多指标与风险护栏。

12) 模型故障降级策略​

代码示例​

if (providerA.down) fallback(providerB || cachedAnswer)

原理讲解​

降级要有优先级链路：主模型 -> 备模型 -> 缓存/规则。

练习​

  • 基础练习：画出你的降级状态机。
  • 进阶练习：设计半开恢复策略。

高频追问​

如何避免“雪崩重试”？​

加熔断、指数退避和全局并发闸门。

面试口述模板​

降级策略目标是“可用优先”，并保证可恢复与可观测。

13) 数据隐私与合规​

代码示例​

text = maskPII(text, ['phone','idCard'])

原理讲解​

输入输出都需脱敏与审计，最小化敏感数据暴露。

练习​

  • 基础练习：列出 5 类需脱敏字段。
  • 进阶练习：设计脱敏前后审计日志。

高频追问​

为什么输出也要做安全扫描？​

模型可能复述敏感信息或生成违规内容。

面试口述模板​

合规是全链路问题：采集、传输、存储、输出都要管。

14) 线上事故复盘方法​

代码示例​

时间线 -> 影响面 -> 根因 -> 修复 -> 防再发

原理讲解​

复盘要从“现象”回到“系统机制”，形成可执行改进项。

练习​

  • 基础练习：写一次假想故障复盘。
  • 进阶练习：给出 3 条防再发行动项。

高频追问​

如何证明改进有效？​

通过回归评测、压测和线上指标对比验证。

面试口述模板​

事故复盘关键是可验证改进，而不是只给结论。

15) AI 项目技术选型怎么答？​

代码示例​

需求约束 -> 方案候选 -> 评测对比 -> 决策与回滚

原理讲解​

面试看的是取舍能力：质量、延迟、成本、复杂度平衡。

练习​

  • 基础练习：给“知识问答”做三方案对比。
  • 进阶练习：写一份选型 ADR。

高频追问​

为什么你选这套，不选另一套？​

用数据（评测/压测/成本）而不是主观偏好回答。

面试口述模板​

技术选型本质是约束下的最优解，并且要可量化、可回滚。
